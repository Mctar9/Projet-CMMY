Classe Circuit

Package : (par défaut)

Représente le panneau principal du circuit logique.
Gère les composants, les fils, les interactions utilisateur et les différents modes d’édition.

Constructeur

public Circuit()

Initialise les écouteurs souris, clavier et configure le panneau.

Méthodes principales

void enableAddingComponent(String type)

Active le mode d'ajout de composant.

void enableConnectingMode()

Active le mode de connexion entre composants.

void enableDeletingMode()

Active le mode de suppression.

protected void paintComponent(Graphics g)

Dessine les fils et composants du circuit.

Méthodes utilitaires

private MemoryComponent getComponent(int x, int y)

Retourne le composant situé à une position donnée.

private Wire getWireAt(int x, int y)

Retourne un fil proche d’un point donné.

private ConnectionPoint findConnectionPoint(int x, int y)

Recherche un point de connexion.

Événements

private void handleMouseClick(MouseEvent e)
private void deleteComponentOrWire(MouseEvent e)
private void addNewComponent(MouseEvent e)
private void handleWireConnection(MouseEvent e)
private void handleKeyPress(KeyEvent e)

Gèrent les interactions avec l’utilisateur (clics, touches...)
