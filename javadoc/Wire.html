Wire Class

public class Wire

Représente un fil reliant deux points dans un circuit logique. Le fil peut être connecté à plusieurs points intermédiaires, mais doit toujours relier une sortie à une entrée. Il gère aussi le dessin graphique du fil.

Package

(aucun package défini)

Class Overview

La classe Wire modélise une connexion entre deux ConnectionPoint dans un circuit numérique :

  • Peut contenir plusieurs connexions intermédiaires
  • Contient une valeur logique ("0", "1", "undefined")
  • Fournit des méthodes pour dessin graphique
  • Vérifie les connexions correctes : sortie → entrée
