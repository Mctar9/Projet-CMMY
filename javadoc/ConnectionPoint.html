ConnectionPoint Class

public class ConnectionPoint

Représente un point de connexion pour un composant logique. Ce point peut être une entrée ou une sortie, et permet d'établir des connexions via des fils dans un circuit logique.

Package

(aucun package défini)

Class Overview

  • Peut être une entrée ou une sortie
  • Permet le dessin graphique du point
  • Détecte la proximité d’un clic (pour l’interaction GUI)
  • Permet le déplacement ou mise à jour de sa position
