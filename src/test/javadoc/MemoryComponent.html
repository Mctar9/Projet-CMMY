Classe MemoryComponent

Composant abstrait représentant un élément logique avec des entrées et des sorties.

Constructeur

public MemoryComponent(int id, String type, int x, int y)

Construit un composant mémoire positionné autour du clic utilisateur.

Méthodes principales

  • moveTo(int x, int y) : Déplace le composant à la position donnée.
  • rotate() : Applique une rotation de 90°.
  • contains(int x, int y) : Vérifie si le point appartient au composant.
  • draw(Graphics g, boolean selected) : Dessine le composant (abstraite).

Points de connexion

  • drawConnectionPoints(Graphics2D) : Dessine les entrées/sorties.
  • getConnectionPointAt(int x, int y) : Retourne un point ciblé.
  • updateConnectionPoints() : Met à jour leur position.
  • initConnectionPoints() : Initialise les points par défaut.
