🎮 Nouveaux raccourcis clavier

➤Suppr : Mode suppression instantané
➤Espace : // Action pour relancer la dernier boutons appuyer si la souris est toujours dessus
➤Ctrl + Z : Annule la dernière action
➤ctrl F: Redimenssioner l'écran
💡 Astuce : Maintenez Alt pendant un déplacement pour dupliquer un composant

🖱️ Drag-and-drop magique

➤ Déposez directement des fichiers .txt dans la zone de travail !
Exemple : Glissez votre circuit préféré depuis le bureau (merci Céline pour cette sorcellerie !)

👻 Prévisualisation des composants

Quand vous sélectionnez un composant :
➤ Une silhouette translucide suit votre souris
➤ Rotation avec R pendant le placement
⚠️ Attention : Ne fonctionne pas avec le café épuisé (testé par Céline)

📐 Adaptation à l'écran

➤ Double-cliquez dans un vide pour centrer le circuit
➤ Tirez les bords de la fenêtre pour redimensionner
🎮 Pro-tip : Ctrl+Molette souris pour zoomer/dézoomer

🚨 Feedback d'erreurs amélioré

➤ Connexion impossible ? Une alerte explique pourquoi
➤ Fils clignotants en violet pour les erreurs logiques
➤ Survolez les composants pour voir leur état
